# Doctor Doom VS Scarlet Witch

## Como funciona ?

En este programa simulamos un pequeño combate dentro del Universo de Marvel, el cuál es totalmente aleatorio, el usuario no realiza ninguna acción (más que la de darle a inicar el programa, obvio).

Al iniciar se pone en contexto antes del inicio del combate (inspirado en el cómic de Secret Wars) para que así tenga sentido.
